# ChatGuard - YouTube Chat Moderation Bot

## Overview

ChatGuard is an AI-powered bot designed to moderate YouTube live chat in real-time. It utilizes generative AI models like OpenAI or Gemini APIs to help maintain a safe and positive environment for viewers.

## ⚠️ Project Status

> **Note:** This repository is currently undergoing a revamp to improve performance, structure, and overall efficiency.

## Features

- **Real-time chat moderation**: Continuously monitors YouTube live chat and automatically moderates messages based on predefined criteria.
- **AI-powered moderation**: Uses advanced NLP models to detect and filter inappropriate content such as spam, hate speech, and profanity.
- **Notification system**: Alerts moderators or administrators about actions taken, allowing manual intervention when needed.
