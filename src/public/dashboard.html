System

Quick start: add Google OAuth credentials, pick Gemini or LM Studio, save, then sign in on Live and start monitoring.

Google OAuth (YouTube)

Required for reading live chat. Create an OAuth client in the Google Cloud Console, enable YouTube Data API v3, and add the redirect URI below.

  1. Add the redirect URI in Google Cloud (must match exactly).
  2. Paste Client ID and secret here, then Save credentials.
  3. Choose Gemini or LM Studio in the next card and save.
  4. Open Live, sign in with Google, then start monitoring.

AI moderation

Google Gemini uses a Google AI API key (cloud). LM Studio runs on this computer: start LM Studio, load a model, turn on the local server (usually http://localhost:1234), then enter the same model id shown in LM Studio. If the primary model fails, the app tries the other provider automatically.

OpenAI-compatible mode is on — turn it off in Advanced to use this dropdown.
